# 1509. Minimum Difference Between Largest and Smallest Value in Three Moves

Given an array `nums`, you are allowed to choose one element of `nums` and change it by any value in one move.

Return the minimum difference between the largest and smallest value of `nums` after perfoming at most 3 moves.

**Example 1:**

```text
Input: nums = [5,3,2,4]
Output: 0
Explanation: Change the array [5,3,2,4] to [2,2,2,2].
The difference between the maximum and minimum is 2-2 = 0.
```

**Example 2:**

```text
Input: nums = [1,5,0,10,14]
Output: 1
Explanation: Change the array [1,5,0,10,14] to [1,1,0,1,1]. 
The difference between the maximum and minimum is 1-0 = 1.
```

**Example 3:**

```text
Input: nums = [6,6,0,1,1,4,6]
Output: 2
```

**Example 4:**

```text
Input: nums = [1,5,6,14,15]
Output: 1
```

**Time:**O\(nlog⁡n\)

 **Space:**O\(1\)

```java
class Solution {
  public int minDifference(int[] nums) {
    final int n = nums.length;
    if (n < 5)
      return 0;

    int ans = Integer.MAX_VALUE;

    Arrays.sort(nums);

    // change nums[0..i) to nums[i] and
    // change nums[n - 3 + i..n) to nums[n - 4 + i]
    for (int i = 0; i <= 3; ++i)
      ans = Math.min(ans, nums[n - 4 + i] - nums[i]);

    return ans;
  }
}
```

```java
class Solution {
    public int minDifference(int[] nums) {
        if (nums.length <= 4) {
            return 0;
        }

        Queue<Integer> minHeap = new PriorityQueue<>(4);
        Queue<Integer> maxHeap = new PriorityQueue<>(4, (o1, o2) -> o2 - o1);
        for (int num : nums) {
            if (maxHeap.size() < 4) {
                maxHeap.offer(num);
            } else if (maxHeap.peek() > num) {
                maxHeap.poll();
                maxHeap.offer(num);
            }

            if (minHeap.size() < 4) {
                minHeap.offer(num);
            } else if (minHeap.peek() < num) {
                minHeap.poll();
                minHeap.offer(num);
            }
        }
        int[] smallest = new int[4];
        int[] largest = new int[4];
        for (int i = 0; i < 4; i++) {
            smallest[3 - i] = maxHeap.poll();
            largest[i] = minHeap.poll();
        }
        int ans = Integer.MAX_VALUE;
        for (int deleteFromLeft = 0; deleteFromLeft <= 3; deleteFromLeft++) {
            ans = Math.min(ans, largest[deleteFromLeft] - smallest[deleteFromLeft]);
        }
        return ans;
    }
}
```
